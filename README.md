# Medical Association Website (Static Version)

Esta é a versão **Front-end Puro** do site da S.P.O.

- **Tecnologia**: HTML5, JavaScript (ES6+), Tailwind CSS (via CDN).
- **Sem Backend**: Não requer Python, Node.js ou Banco de Dados para rodar visualmente.
- **Persistência**: Os dados do "Admin" são salvos no navegador do usuário (LocalStorage) para demonstração.

## Como Usar

Basta abrir o arquivo `index.html` no seu navegador.

Ou, se preferir usar um servidor estático simples:
```bash
python3 -m http.server 8000
```
Acesse: `http://localhost:8000`
